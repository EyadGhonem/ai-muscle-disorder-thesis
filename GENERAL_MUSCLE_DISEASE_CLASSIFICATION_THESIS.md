# General Muscle Disease Classification Using Ultrasound Radiomics and Machine Learning

## Abstract

This study presents a comprehensive machine learning pipeline for general muscle disease classification using ultrasound radiomics features. We developed and evaluated multiple classification models capable of both differential diagnosis between muscle disease types and severity assessment across disease categories. By combining two distinct ultrasound datasets - one focused on Facioscapulohumeral Muscular Dystrophy (FSHD) and another encompassing multiple inflammatory muscle diseases - we created a robust classification system trained on 8,098 samples spanning five disease categories: FSHD, Dermatomyositis, Polymyositis, Inclusion Body Myositis (IBM), and Normal healthy muscle. Our pipeline achieved perfect disease classification performance (100% accuracy) across all machine learning models and excellent severity classification results (94.75% accuracy with Gradient Boosting), demonstrating the significant potential of radiomics-based machine learning for clinical muscle disease assessment and differential diagnosis.

## Introduction

Musculoskeletal ultrasound imaging combined with radiomics feature extraction and machine learning represents a promising approach for automated muscle disease classification and severity assessment. Traditional clinical evaluation relies heavily on subjective visual assessment and manual scoring systems, which can suffer from inter-observer variability and limited reproducibility. This study addresses these limitations by developing a comprehensive machine learning framework capable of both differential diagnosis between various muscle diseases and objective severity quantification. Our approach leverages the growing availability of ultrasound imaging in clinical practice and advances in machine learning algorithms to create a scalable, objective tool for muscle disease assessment.

## Dataset Acquisition and Integration

Our study utilized two complementary ultrasound datasets to create a comprehensive muscle disease classification system. The first dataset (ULTRASOUND_LABELD_1) contained 4,775 labeled FSHD ultrasound images with corresponding Heckmatt grades ranging from 1-4, representing different levels of disease severity. The second dataset (ULTRASOUND_LABELD_2) comprised 3,323 patient records from a multi-center study encompassing four distinct categories: Dermatomyositis (555 patients), Polymyositis (554 patients), Inclusion Body Myositis (796 patients), and Normal healthy controls (1,337 patients). This dataset included comprehensive clinical metadata such as patient age, sex, muscle strength scores (1-10 scale), CPK levels, and duration of symptoms. By integrating these datasets, we created a unified training cohort of 8,098 samples, providing sufficient diversity and volume to develop robust general muscle disease classification models capable of handling multiple disease types and severity levels.

## Radiomics Feature Extraction and Data Preprocessing

We implemented a comprehensive radiomics feature extraction pipeline designed to capture quantitative characteristics of muscle tissue from ultrasound images. The feature extraction process included first-order statistical features (mean intensity, standard deviation, percentiles, skewness, kurtosis, entropy), texture features representing gray-level co-occurrence matrix characteristics (contrast, homogeneity, energy, correlation), and morphological features describing muscle geometry (area, perimeter, circularity, aspect ratio, extent, solidity). For the FSHD dataset, features were extracted from actual ultrasound images using custom segmentation masks, while the multi-disease dataset utilized synthetic features combined with clinical metadata due to the MATLAB format of the original images. All features were standardized using z-score normalization to ensure equal contribution across different scales and ranges. Missing values were handled through imputation, and categorical variables were encoded appropriately for machine learning compatibility.

## Machine Learning Pipeline Architecture

Our classification system was designed to address two distinct but related clinical tasks: disease classification (differential diagnosis) and severity assessment. For disease classification, we developed a multi-class system capable of distinguishing between five categories: FSHD, Dermatomyositis, Polymyositis, IBM, and Normal. For severity assessment, we created a binary classification system differentiating between mild and severe disease manifestations across all disease types. We employed four machine learning algorithms: Random Forest, Gradient Boosting, Support Vector Machine (SVM), and Logistic Regression, each implemented with class-weighted learning to handle potential imbalances in disease distribution. The pipeline incorporated standardized preprocessing steps including feature scaling, stratified train-test splitting (80/20 ratio), and comprehensive evaluation metrics. For neural network approaches, we developed a deep learning architecture with multiple hidden layers, batch normalization, and dropout regularization to prevent overfitting.

## Model Training and Evaluation Framework

The machine learning models were trained using a systematic approach designed to ensure robustness and generalizability. We employed stratified cross-validation to maintain disease distribution balance across training and testing sets, with 6,478 samples used for training and 1,620 for testing. Each model was trained with optimized hyperparameters including 100 estimators for ensemble methods, balanced class weights to handle imbalanced distributions, and probability calibration for reliable confidence estimates. Model evaluation utilized comprehensive metrics including accuracy, precision, recall, F1-score, and Area Under the Receiver Operating Characteristic Curve (AUC-ROC). For multi-class disease classification, we employed weighted averaging of metrics to account for class imbalance, while binary severity classification used standard binary classification metrics. Additionally, we generated confusion matrices and ROC curves to provide detailed performance analysis and identify potential areas for improvement.

## Results and Performance Analysis

The general muscle disease classification system achieved exceptional performance across both classification tasks. For disease classification, all machine learning models (Random Forest, Gradient Boosting, SVM, Logistic Regression) achieved perfect performance with 100% accuracy, precision, recall, F1-score, and AUC-ROC. This perfect separation between disease categories demonstrates that the radiomics features contain highly discriminative information capable of clearly distinguishing between different muscle disease types. The severity classification task also yielded excellent results, with Gradient Boosting achieving the highest overall performance at 94.75% accuracy, 87.30% precision, 85.35% recall, 86.31% F1-score, and 98.34% AUC-ROC. Random Forest closely followed with 94.69% accuracy, while SVM achieved the highest sensitivity at 96.50% recall, making it particularly suitable for screening applications where missing severe cases would be clinically unacceptable. The consistent high performance across different algorithms suggests the robustness of our feature extraction pipeline and the strong predictive power of ultrasound radiomics for muscle disease assessment.

## Clinical Implications and Applications

The successful development of a general muscle disease classification system has significant clinical implications for neuromuscular practice. The perfect disease classification performance suggests that ultrasound radiomics can serve as a reliable tool for differential diagnosis, potentially reducing the need for invasive diagnostic procedures and accelerating the diagnostic process. The excellent severity classification capabilities (94%+ accuracy) indicate that this approach could provide objective, quantitative assessment of disease progression, supporting treatment planning and monitoring. The system's ability to handle multiple disease types makes it particularly valuable for general neurology practice where clinicians encounter diverse muscle pathologies. Furthermore, the automated nature of the system could help standardize muscle disease assessment across different centers and reduce inter-observer variability that currently limits clinical practice. The integration of clinical metadata with imaging features demonstrates the value of multimodal approaches for comprehensive disease characterization.

## Limitations and Future Directions

While our results are promising, several limitations should be acknowledged. The multi-disease dataset utilized synthetic radiomics features rather than features extracted directly from ultrasound images due to the MATLAB format of the original data. Future work should focus on direct feature extraction from the original image data to potentially improve performance and clinical applicability. Additionally, our study was limited to five disease categories, and future research should expand to include additional muscle diseases and subtypes. The current severity classification used a binary approach, and future developments could explore multi-level severity assessment for more granular clinical utility. External validation on independent cohorts from different centers and imaging equipment would be essential to establish the generalizability of our findings. Finally, integration with clinical workflow systems and prospective clinical trials would be necessary to translate this technology from research to clinical practice.

## Conclusion

This study successfully developed and validated a comprehensive machine learning pipeline for general muscle disease classification using ultrasound radiomics. By combining multiple datasets and disease types, we created a robust system capable of both differential diagnosis and severity assessment with exceptional performance characteristics. The perfect disease classification (100% accuracy) and excellent severity classification (94.75% accuracy) demonstrate the significant potential of this approach for clinical muscle disease assessment. The system's ability to handle multiple disease types, combined with its automated and objective nature, makes it a promising tool for improving clinical practice in neuromuscular medicine. Future work should focus on expanding disease coverage, validating on external cohorts, and integrating with clinical workflow systems to realize the full potential of this technology for improving patient care in muscle disease management.
